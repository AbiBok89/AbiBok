## Christmas Shop. Part-1 – Fixed Layout

### Основное описание задачи и требований

- [Подробное описание](christmas-shop.md)

Необходимо создать страницы **Home** и **Gifts** в соответствии с  
[Дизайном во Figma](https://www.figma.com/design/zTB01BwWZVoXYK5atH3eZT/Cristmas-Shop) – только  
`Home / w1440` и `Gifts / w1440`. Ширина контента 1440 px не должна изменяться при ресайзе окна браузера.

## Рабочий процесс

1. Выполнить задачу в приватном репозитории школы.  
2. Создать новую ветку `christmas-shop` от ветки `main`. Внутри неё создать папку `christmas-shop` и разместить весь код там.  
3. Завершить работу, убедившись, что она соответствует [Критериям кросс‑чек](#критерии-кросс-чека).  
4. Развернуть проект на gh‑pages (создать и слить PR из ветки `christmas-shop` в ветку `gh-pages`). Титул Pull Request можно выбрать произвольно, описание не требуется.  
5. Открыть Pull Request из ветки `christmas-shop` в ветку `main`, дать ему название по имени задачи и заполнить шаблон описания PR согласно [руководству](https://docs.rs.school/#/pull-request-review-process?id=Требования-к-pull-request-pr). Слияние не требуется.  
6. В разделе «Cross‑Check Submit» приложения RS App (https://app.rs.school/) отправить ссылку на развернутый сайт.  
7. После окончания дедлайна начнётся фаза кросс‑чека – 3 дня. Чтобы заработать баллы, необходимо просмотреть все проекты, назначенные вам для кросс‑чекинга, и в разделе «Cross‑Check Review» приложения RS App отправить результаты проверки.

## Специфика проверки макета

- Допускается отклонение до **10 px** горизонтально/вертикально, при условии сохранения визуальной согласованности с дизайном.
- Используйте Chrome‑расширение [PerfectPixel](https://chrome.google.com/webstore/detail/perfectpixel-by-welldonec/dkaagdgjmgdmbnecmcefdhjekcoceebi?hl=ru) для проверки выравнивания.
- При работе с PerfectPixel установите масштаб 1 и убедитесь, что система/браузер выставлены в 100 % масштаб.
- Если разрешение экрана > 1440 px – достаточно центрировать макет по горизонтали (или вручную привязать к левому верхнему направляющему).
- При ширине окна ≤ 1440 px используйте режим responsive в Chrome DevTools.
- Для текста проверяйте только высоту шрифта; отклонения в ширине слов и межбуквенном интервале не считаются ошибками, если выбран правильный шрифт с указанными свойствами.

## Как отправить

1. Разверните ваш сайт (https://github.com/rolling-scopes-school/<your‑repo>/christmas-shop) на gh‑pages.
2. Откройте приложение RS App и перейдите в **Cross‑Check: Submit**.
3. Выберите задачу «Christmas Shop. Part-1 – Fixed Layout» из выпадающего списка.
4. Введите URL развернутого сайта в поле **Solution URL** и нажмите кнопку **Submit**.

## Рекомендации по сдаче

- Сдавайте как можно раньше, после появления возможности в RS App.
- В приватных репозиториях не нужно делиться ссылками на репозиторий – рецензенты кросс‑чека не имеют доступа к ним.  
- Убедитесь, что развернутый сайт корректно работает в режиме incognito.
